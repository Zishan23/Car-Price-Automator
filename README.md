# Car Price Automator

An intelligent system for predicting car prices using machine learning and data analysis.

## Project Structure

```
car-price-automator/
├── backend/          # Node.js API server
├── frontend/         # React frontend application
└── README.md         # This file
```

## Features

- **Price Prediction**: ML-powered car price estimation
- **Data Ingestion**: Automated data collection from multiple sources
- **API**: RESTful API for price predictions
- **Dashboard**: Interactive web interface for price analysis

## Getting Started

### Backend Setup

```bash
cd backend
npm install
npm start
```

### Frontend Setup

```bash
cd frontend
npm install
npm start
```
